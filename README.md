# 디플루이드 코딩테스트 과제

## 주요 기능
- 프로필 카드 임의 배치
- API를 이용해 임의의 배경 이미지 삽입
- 이메일 유효성 검사
- 카드 데이터 정의, 필터 기능 구현

## 사용 기술
- React (Hooks 기반)
- SCSS (CSS Modules)
- Swiper (이미지 슬라이더)
